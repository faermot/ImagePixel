# ImagePixel - Pixelizer
_Это простая программа на WPF, которая позволяет пикселизировать изображения с настройкой размера пикселей._

<img width="809" height="617" alt="image" src="https://github.com/user-attachments/assets/143d06dc-482f-4a06-8bc0-2e9d87acf0c0" />

---

## Характеристики

- Загрузка изображений в форматах PNG, JPEG.
- Настройка размера пикселя (до 128 пикселей).
- Предпросмотр результата в реальном времени.
- Сохранение результата в выбранном формате.
- Простой и интуитивно понятный интерфейс (WPF).

## Скачать

Готовый `.exe` без необходимости устанавливать .NET - на странице [Releases](https://github.com/faermot/ImagePixel/releases).

## Сборка из исходников
```bash
git clone https://github.com/faermot/ImagePixel.git
cd ImagePixel
dotnet build
```

Требуется .NET 8 и Windows.
